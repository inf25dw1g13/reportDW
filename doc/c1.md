# C1 : Food Delivery API

## Description
Este trabalho tem como objetivo fazer uma interação entre uma aplicação, api, que através de métodos post, get, put e delete, interage com a sua database, simulando uma plataforma de entregas de comida.

O projeto utiliza uma api que apresenta em local host a simulação da criação do(s) pedido(s), obtenção de informação sobre o(s) mesmo(s), bem como a deleção. Além disso, utiliza-se uma base de dados MariaDB para estrutura e armazenamento de dados relacionados às entregas com as seguintes tabelas:
- Restaurantes, Pratos, Clientes, Pedidos, Ingredientes e Entregas.

Além disso, inclui um MaxScale como middleware e 2 bases de dados que funcionam como réplicas da original.
