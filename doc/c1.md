# C1 : Food Delivery API

## Descrição
Este trabalho tem como objetivo fazer uma interação entre uma aplicação, api, que através de métodos post, get, put e delete, interage com a sua database, simulando uma plataforma de entregas de comida.

O projeto consiste-se numa api que faz simulação da criação do(s) pedido(s), obtenção de informação sobre o(s) mesmo(s), bem como a deleção. Além disso, utiliza-se uma base de dados MariaDB para estrutura e armazenamento de dados relacionados às entregas com as seguintes tabelas:
- Restaurantes, Pratos, Clientes, Pedidos, Ingredientes e Entregas.

Além disso, inclui um MaxScale como middleware e 2 bases de dados que funcionam como réplicas da original, sendo que estas são apenas para leitura (MaxScale ajuda no controlo para as base de dados relativamente a se é método de escrita - CRUD -, primeira BD, ou somente de leitura, réplica). Ou seja, o MaxScale, através do tipo de pedido, determina qual base de dados vai aceder e se vai (ou não) alterar.
